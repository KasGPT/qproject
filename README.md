# NOVELL: The World’s First Quantum AI DAG Agent Protocol

![Build Status](https://img.shields.io/badge/build-passing-brightgreen)
![License](https://img.shields.io/badge/license-MIT-blue)
[📖 Documentation](https://github.com/KasGPT/qproject/tree/main/docs)

---

NOVELL is a decentralized agent-based task execution engine that combines:
- ✅ DAG Task Automation (KASTASK)
- 🧠 Multi-Agent AI Orchestration (LangChain-style)
- 🔐 Wallet Integration (Kaspa + MetaMask)
- 🧾 Governance Voting + Agent Reputation (Devnet-ready)
- 🪙 Stablecoin, ETF, DeFi Tokenization (via Agents)

---

## 🚀 Quickstart

```bash
pip install -r requirements.txt
uvicorn core.main:app --reload
```

Then visit: http://localhost:8000
